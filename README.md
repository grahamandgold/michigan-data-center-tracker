# Michigan Data Center Tracker

Independent public-source tracking of Michigan data center projects, public meetings, moratoria, power, water and policy.

- `index.html` — homepage
- `content-data.js` — editorial content and integration settings
- `map.html` / `map.js` — sourced project and policy map
- `methodology.html` — standards, confidence labels and corrections
- `privacy.html` — newsletter and contact privacy notice
- `EDITORIAL-GUIDE.md` — Grok, Mailchimp, X and Reddit update instructions

Routine editorial updates should be made in `content-data.js`, not in the page layout.
